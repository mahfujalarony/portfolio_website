# Mahfuj Alam - Developer Portfolio

A modern, responsive, and SEO-optimized personal portfolio website built with Next.js 16.

## 🚀 Tech Stack

- **Framework**: Next.js 16 with Turbopack
- **Language**: TypeScript
- **UI Library**: React 19
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **Deployment**: Vercel

## 🔧 Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.copy` to `.env.local` and fill in the required values.

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Customization

| Section           | File Location             |
| ----------------- | ------------------------- |
| Personal Info     | `config/site.ts`          |
| Skills            | `config/skills.ts`        |
| Projects          | `config/projects.ts`      |
| Experience        | `config/experience.ts`    |
| Social Links      | `config/socials.ts`       |
